IT Keyboard Shortcuts

  • CTRL + L: clear all notes in nu.
  • CTRL + C: stop or clear any running process.
  • Q: quit when running quiver in the terminal.
  • CTRL + K: paste a website into Microsoft Teams.
  • CTRL + ~: open the terminal in Zed.
  • cd: change into a folder (use Tab to autocomplete names); use cd ~ to go back out of all folders.
  • CTRL + ,: open the settings page in the terminal.
  • CTRL + Shift + Esc: open Task Manager to check CPU and RAM usage.
  • code .: open the current folder in Visual Studio Code.
  • Windows + V: open emoji and paste history.

Salesforce Use

  • In Salesforce, Setup->Sandbox is how you access the multiple sandboxes for testing code. The sandbox has a deleted folder and can somewhat track what was done.
  • Salesforce Inspector Reloaded lets you write SOQL queries (just click data export).
  • Get the Salesforce extension for your browser to allow an easier Salesforce experience?
  • Setup-> Storage is the organization's storage.
  • JotForm was used to build the AHS application.
    1. JotForm Immediately changes whatever is on the frontend when a change is made to the form.
    2. Ensure you have a JotForm account. Be careful when making changes!
  • Every day, a daily asset activator checks if the start date is the next day. If so, status is sent to active.
  • Fifteen minutes later, the daily asset deactivator checks if any assets have deactivated, meaning they are set to inactive.

Software Developer Setup

  • Install a Chatbot on your computer?
  • If on a Windows computer, install Windows Subsystem for Linux (WSL).
  • Install Ubuntu, Quiver, and Nu:
    • Resources for Nu learning: Nu Commands and Github Nu
    • Nu-client-tools with-spinner shows a cool spinner while loading followed by a checkmark when installed.
    • $ is how nu recognizes a variable ($today instead of today).
    • | is the pipeline operator which sends items from one pipeline into another. You can use it to select & put a space between the columns.
    • $"" with parenthesis around variable functions like an f-string.
    • A record in nu is the same as a dictionary in Python.
    • The reduce function outputs a record, not a list or table.
    • SOQL is the Salesforce Object Query Language.
    • --all queries everything. sf query sends a query to Salesforce and then returns the result.
    • It may be good to use the terminal to test the code without running the entire program. The following items are in the terminal:
      • Use nu-salesforce loads the library* load-env-file pulls the nu file into objects
      • $env.USERNAME loads the username
      • Then use sf login to login to the salesforce system
      • sf whoami: ensures you are in the sandbox
      • nu-shell works great with json (open is the command to open the file)
    • Gitignore files do not go on Github
    • def main is the main function.
    • You can use both Python and nu-shell in the same system.
    • let, const or any other immutable variables require a $.
  • You can use Zed as an editor.
  • In order to install Quiver, use qv run nu and qv to ensure it is there.
  • Here is the Ubuntu terminal screen: Ubuntu Terminal
  • To open the config env file, use the config env(tab) and it will pop up a little screen. Nu Config
  • If you are going to be using the terminal and Ubuntu, create a new external client app so that you can have your own key.
  • If you want to go to Zed, just type Zed once in the folder.
